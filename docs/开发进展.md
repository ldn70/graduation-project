# 开发进展（实时）

## 当前完成
- 后端：
  - Django + DRF + JWT 基础框架完成。
  - 已实现接口：
    - 用户：`/api/users/register`、`/api/users/login`、`/api/users/profile`、`/api/users/delete`
    - 搜索：`/api/jobs/search`
    - 推荐：`/api/recommend/jobs`
    - 技能：`/api/skills/demand`、`/api/skills/match`
    - 薪资：`/api/salary/predict`
    - 趋势：`/api/trends/jobs`
    - 简历：`/api/resume/generate`、`/download/<filename>`
  - 迁移与样例数据脚本可用：`manage.py migrate`、`manage.py seed_jobs`
  - 数据导入升级（`[DONE]`）：
    - `manage.py import_jobs_csv` 支持 CSV -> MySQL。
    - 新增可选 CSV -> MongoDB 同步（`--sync-mongo`，`job_texts` 集合）。
  - 推荐算法升级（`[DONE]`）：
    - 新增离线训练命令：`manage.py train_recommend_model`。
    - 推荐链路升级为 Hybrid：TF-IDF + CF + LightFM（无 LightFM 时自动 SVD 降级）。
    - `/api/recommend/jobs` 优先使用训练模型，未训练时自动回退 baseline。
  - 薪资预测升级（`[DONE]`）：
    - 新增离线训练命令：`manage.py train_salary_model`。
    - `/api/salary/predict` 优先使用 XGBoost 模型推理并返回 SHAP 解释。
    - 模型不可用时自动回退到 baseline 估算逻辑，接口不降级。
  - 趋势模型升级（`[DONE]`）：
    - 新增离线训练命令：`manage.py train_trend_model`。
    - `/api/trends/jobs` 支持 `time_range=month|quarter|year` 聚合。
    - 月维度 `forecast=true` 优先使用 Hybrid（Prophet + LSTM 代理）推理，模型不可用时自动回退 baseline。
  - 自动化测试升级（`[DONE]`）：
    - 新增后端 API 回归测试：`users/jobs/resumes/analysis` 核心接口覆盖。
    - 已验证测试命令：`manage.py test`（当前 19 项全部通过）。
- 前端：
  - Vue3 页面已接入上述全部核心接口。
  - 已包含 ECharts 可视化（地域分布 + 趋势图）。
  - 前端测试与展示升级（`[DONE]`）：
    - 新增 Vitest 关键流程测试（登录/搜索/趋势交互）。
    - 趋势模块新增 `time_range` 选择、`model_info` 展示与预测区间可视化（上界/下界）。
- 工程化：
  - `docker-compose.yml`、前后端 `Dockerfile` 已创建。
  - 根目录 `README.md` 可指导启动。
