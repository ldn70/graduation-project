# 文档导航

本目录用于统一管理项目重建过程中的开发与交付文档。

## 文件列表
- `执行计划.md`：两天重建排期、里程碑与风险预案。
- `功能模块.md`：七大核心功能 + 可视化模块的功能说明。
- `接口清单.md`：后端 API 路径、方法、参数与返回约定。
- `数据库设计.md`：MySQL/MongoDB 结构、字段与索引建议。
- `测试计划.md`：功能、安全、性能测试范围与验收目标。
- `数据导入说明.md`：CSV 数据导入命令与字段映射说明。

## 使用顺序（建议）
1. 先读 `执行计划.md` 明确两天节奏。
2. 按 `功能模块.md` 与 `接口清单.md` 并行开发前后端。
3. 按 `数据库设计.md` 建表和初始化数据。
4. 用 `测试计划.md` 执行回归和交付验收。
