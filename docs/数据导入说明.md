# 数据导入说明

本项目已支持从 CSV 快速导入招聘数据到 MySQL `jobs` 表，并可选同步到 MongoDB `job_texts` 集合。

## 1. 命令
在 `backend` 目录执行：

```powershell
..\.venv\Scripts\python manage.py import_jobs_csv --file D:\GraduationProject\backend\data\sample_jobs.csv --truncate
```

如需同时同步到 MongoDB：

```powershell
..\.venv\Scripts\python manage.py import_jobs_csv --file D:\GraduationProject\backend\data\sample_jobs.csv --truncate --sync-mongo --mongo-truncate
```

## 2. 常用参数
- `--file`：CSV 文件路径（必填）
- `--encoding`：文件编码，默认 `utf-8-sig`
- `--delimiter`：分隔符，默认 `,`
- `--truncate`：导入前清空已有职位数据
- `--limit`：仅导入前 N 条
- `--sync-mongo`：将导入后的职位长文本同步到 MongoDB
- `--mongo-uri`：MongoDB 连接串，默认读取环境变量 `MONGO_URI`
- `--mongo-db`：MongoDB 数据库名，默认 `graduation_project`
- `--mongo-collection`：MongoDB 集合名，默认 `job_texts`
- `--mongo-timeout-ms`：Mongo 连接超时毫秒数，默认 `3000`
- `--mongo-truncate`：同步前清空 Mongo 目标集合（需与 `--sync-mongo` 一起使用）

## 3. 支持字段（可中英文字段名）
- `title` / 职位名称 / 岗位名称
- `company` / 公司名称
- `salary` / 薪资范围
- `education` / 学历要求
- `experience` / 工作经验
- `skills_required` / 技能要求
- `description` / 职位描述
- `location` / 工作地点 / 城市
- `industry` / 行业
- `publish_time` / 发布时间
- `job_url` / 职位链接

## 4. 模板文件
- 示例文件：[sample_jobs.csv](D:/GraduationProject/backend/data/sample_jobs.csv)

## 5. 环境变量（Mongo 同步可选）
`backend/.env` 可配置：

```env
MONGO_URI=mongodb://127.0.0.1:27017/
MONGO_DB=graduation_project
MONGO_JOB_TEXTS_COLLECTION=job_texts
MONGO_TIMEOUT_MS=3000
```

## 6. 导入后验证
可通过以下接口验证：
- `GET /api/jobs/search?keyword=Python`
- `GET /api/skills/demand?top_n=10`
- `GET /api/trends/jobs?forecast=true`
