# 接口清单（MVP）

## 1. 统一约定
- Base URL：`/api`
- 认证方式：JWT Bearer Token
- 返回格式：
```json
{
  "success": true,
  "message": "ok",
  "data": {},
  "statusCode": 200,
  "code": "OPTIONAL_ERROR_CODE"
}
```
- 说明：
  - 成功响应通常不返回 `code` 字段。
  - 失败响应可返回 `code` 字段，作为前端稳定映射依据。
  - 详细规范见 `错误码规范.md`。

## 2. 用户管理

## 2.1 注册
- 方法：`POST`
- 路径：`/api/users/register`
- 请求体：
```json
{
  "username": "string",
  "password": "string",
  "name": "string",
  "education": "string",
  "skills": "Python,Django",
  "experience": "string"
}
```

## 2.2 登录
- 方法：`POST`
- 路径：`/api/users/login`
- 请求体：
```json
{
  "username": "string",
  "password": "string"
}
```
- 响应关键字段：`accessToken`、`refreshToken`

## 2.3 更新资料
- 方法：`PUT`
- 路径：`/api/users/profile`
- 请求头：`Authorization: Bearer <accessToken>`

## 2.4 注销账户
- 方法：`DELETE`
- 路径：`/api/users/delete`
- 请求头：`Authorization: Bearer <accessToken>`

## 3. 招聘搜索

## 3.1 搜索职位
- 方法：`GET`
- 路径：`/api/jobs/search`
- Query 参数：
  - `keyword`
  - `education`
  - `salary_min`
  - `salary_max`
  - `page`（默认 1）
  - `per_page`（默认 10）

## 4. 智能推荐

## 4.1 获取推荐列表
- 方法：`GET`
- 路径：`/api/recommend/jobs`
- 请求头：`Authorization: Bearer <accessToken>`
- Query 参数：`limit`（默认 10）

## 5. 技能分析

## 5.1 技能需求分析
- 方法：`GET`
- 路径：`/api/skills/demand`
- Query 参数：
  - `industry`
  - `job_title`
  - `top_n`（默认 20）

## 5.2 技能匹配度分析
- 方法：`GET`
- 路径：`/api/skills/match`
- 请求头：`Authorization: Bearer <accessToken>`
- Query 参数：`job_id`

## 6. 薪资预测

## 6.1 薪资区间预测
- 方法：`POST`
- 路径：`/api/salary/predict`
- 请求体：
```json
{
  "job_title": "Python开发",
  "education": "本科",
  "experience": "3-5年",
  "skills": ["Python", "Django", "MySQL"],
  "city": "北京",
  "industry": "互联网"
}
```

## 7. 趋势分析

## 7.1 岗位趋势查询
- 方法：`GET`
- 路径：`/api/trends/jobs`
- Query 参数：
  - `industry`
  - `job_title`
  - `time_range`（`month|quarter|year`）
  - `forecast`（`true|false`）

## 8. 简历生成

## 8.1 生成简历
- 方法：`POST`
- 路径：`/api/resume/generate`
- 请求头：`Authorization: Bearer <accessToken>`
- 请求体：
```json
{
  "format": "txt"
}
```

## 8.2 下载简历
- 方法：`GET`
- 路径：`/download/<filename>`
- 返回：文件流
